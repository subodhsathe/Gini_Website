# Images for Gini's Website

Add your beautiful photos and images here:

## How to add images:

1. **Save your images** to this folder (`public/images/`)
2. **Supported formats**: JPG, PNG, GIF, WebP
3. **Recommended sizes**:
   - Hero background: 1200x500px
   - Profile photo: 400x400px
   - Gallery images: 800x600px

## Example file names:
- `hero-background.jpg` - Main hero section image
- `gini-performing.jpg` - Performance photo
- `gini-studio.jpg` - Studio photo
- `gini-portrait.jpg` - Portrait photo

## To use in the website:
In App.js, update the image src to:
```
/images/your-image-name.jpg
```

For example:
```
src="/images/gini-performing.jpg"
```
